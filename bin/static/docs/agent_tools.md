## 🔍 WEB 搜索引擎

### 功能特点
- **多源聚合**：集成十余个开源/免费/收费搜索引擎
- **智能排序**：基于大型深度学习模型（BGE-M3）重新排序搜索结果
- **质量过滤**：内置分类器过滤虚假信息和政治敏感内容
- **反幻觉**：为用户提供可核实的依据，有效克服大语言模型幻觉问题

### 技术优势
相比传统搜索引擎，我们的系统能够：
- 避免商业付费导致的不相关结果前置问题
- 提供更精准、更相关的搜索结果
- 确保信息真实性和可靠性

## 🔍 WEB Search Engine

### Features
- **Multi-source Aggregation**: Integrates over ten open-source/free/paid search engines
- **Intelligent Ranking**: Re-ranks search results based on large deep learning models (BGE-M3)
- **Quality Filtering**: Built-in classifier filters out false information and politically sensitive content
- **Anti-hallucination**: Provides verifiable evidence for users, effectively overcoming large language model hallucination issues

### Technical Advantages
Compared to traditional search engines, our system can:
- Avoid irrelevant results being prioritized due to commercial payments
- Provide more accurate and relevant search results
- Ensure information authenticity and reliability
